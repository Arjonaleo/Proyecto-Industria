# Config

Plantillas versionables para configuracion de servidor, IIS, logging y parametros de ejecucion.

Usar archivos de ejemplo o plantilla, sin valores reales de ambientes. Los archivos versionables deben usar alguno de estos patrones:

- `*.template.*`
- `*.example.*`
- `README.md`

No subir passwords, connection strings reales, tokens, certificados privados ni archivos locales o productivos.
