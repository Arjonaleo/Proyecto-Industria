# Scripts

Directorio para automatizaciones operativas del proyecto SGA.

- `build/`: scripts de compilacion y empaquetado.
- `deploy/`: scripts de despliegue para ambientes Dev, QA, UAT y produccion.
- `database/`: scripts operativos para SQL Server que no sean migraciones versionadas.

No subir credenciales, connection strings reales ni archivos `*.local.*`.
